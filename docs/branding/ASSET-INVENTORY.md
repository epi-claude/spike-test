## Logo Usage Guidelines

### When to Use Each Variation

#### Full Color Logos
- **Primary use:** Digital applications (website, presentations, digital marketing)
- **Backgrounds:** White, light gray, or pale blue
- **Format:** PNG with transparency

#### Black Logos
- **Primary use:** Print materials, one-color applications
- **Backgrounds:** White or very light backgrounds
- **Format:** PNG or convert to vector for print

#### Vertical vs. Horizontal
- **Horizontal:** Website headers, email signatures, wide banners
- **Vertical:** Social media, app icons, square/tall layouts
- **Square:** Profile pictures, avatars, compact spaces

### Minimum Size Requirements
- **Horizontal logo:** Minimum 200px wide for web, 1.5 inches for print
- **Vertical logo:** Minimum 150px wide for web, 1 inch for print
- **Icon only:** Minimum 32px for web (favicon), 0.5 inches for print

### Clear Space
Maintain clear space around logo equal to the height of the lightbulb icon on all sides.

### Do Not
- ‚ùå Stretch or distort the logo
- ‚ùå Change the colors (except approved black/white versions)
- ‚ùå Separate the icon from the text
- ‚ùå Add effects (drop shadows, gradients, etc.)
- ‚ùå Place on busy or low-contrast backgrounds
- ‚ùå Rotate the logo

## File Format Recommendations

### Current State
All logos are currently in **PNG** format with transparency.

### Recommended Additions
For maximum flexibility, consider creating:

1. **SVG versions** (vector)
   - Infinitely scalable
   - Small file size
   - Perfect for web use
   - Convert from PSD or recreate in Illustrator/Figma

2. **ICO format** (favicon)
   - Convert `episolve-favicon.png` to `.ico`
   - Include multiple sizes: 16x16, 32x32, 48x48

3. **WebP versions** (optimized)
   - Smaller file size than PNG
   - Better web performance
   - Maintain as fallback to PNG
